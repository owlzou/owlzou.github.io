将主题 Cafe 升级到了 v2 ☕

2021-11-21
日志

今年年初写的这个的主题Cafe（写了个新主题 Cafe☕），现在改造了下成了 v2 版本 😆。

主要的改动

  1. 版式由左右变成了上下

    虽然喜欢将导航放在左边，但正文不放中间总觉得脖子是歪的（？），所以把导航放回了上面。

  2. 将 TOC 放在了猫的对话气泡里

    这其实是一开始就想做的事情……不然在左下角放猫做什么呢。

  3. 新增了页面将目录和标签放在了一起

  4. 新的主页

    之前是用了些 Tick 做成和 Hexo 差不多的“主页就是博客第一页”，现在是单独的主页。

    添加了头图和网站图标支持。

更多的短代码

新增了两个短代码：

  1. 模仿文档里常有的 Tips 块。

    用不用得上存疑。

信息说明块
警告块
{% tips(type="info") %}
信息说明块
{% end %}

{% tips(type="warning") %}
警告块
{% end %}

  1. 带文字说明的图像块。

    括在<figure>标签内。

{{img(src="pxWeZ2xFFa.png",caption="就像这样可以写图片标题")}}

{{img(src="pxWeZ2xFFa.png",caption="还可以定义宽度",width="120")}}
